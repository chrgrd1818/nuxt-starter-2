### App engineering PRINCIPLES:

1. hight Separation of Concerns (SoC) and modularity
2. Three-Layer Separation:
   - UI Layer: Views must be essentially presentational. They meant to display data and trigger props callbacks. No direct calls to DB. Default Styling must be Scoped Semantic grouping of styles, and repetitive elements (eg cards, buttons...) should be setb as reusable components.
   - State/Logic Layer
   - Backend Server Layer : structured server with dedicated api routes, ready for connecting to Supabase or using local dataset.
3. Strict Typing: Eliminate all occurrences of 'any'. Enforce clear TypeScript interfaces for all frontend props and backend API payloads.
